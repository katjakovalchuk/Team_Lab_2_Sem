To set up docker, do
```bash
$ docker build -t reveal-wrapper . 
$ docker run revear-wrapper
```
Of course, you have to install it first
Should you want to add environment variables to docker, do
```bash
$ docker run  --env-file ./.env reveal-wrapper
```
